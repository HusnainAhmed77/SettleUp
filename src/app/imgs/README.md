# Images Folder Structure

This folder contains images for the homepage.

## Usage
Place homepage-specific images here such as:
- Hero section images
- Feature card images
- Testimonial avatars
- CTA section images
- Any other homepage-specific graphics

## Image Naming Convention
- Use lowercase with hyphens: `hero-image.png`
- Be descriptive: `feature-track-balances.svg`
- Include size if multiple versions: `logo-small.png`, `logo-large.png`
